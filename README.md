# Khan Interactive Data Analysis

目标驱动的 Excel/CSV 复杂数据清洗与分析 Skill。

## Workflow

```text
只读数据侦察
  → 数据能力与分类目标首页
  → 选择候选或多轮共创自定义目标
  → 确认目标数据范围与质量处置
  → 生成独立定向清洗文件
  → 分析并保存结果
  → 返回首页继续其他目标
```

主要能力：

- 每个适用类别生成 3 至 5 个高信息量目标候选。
- 展示可支持结论、不可支持结论、数据缺口和清洗成本。
- 通过多轮交流理解和完善用户自定义目标。
- 一个数据工作区管理多个独立目标。
- 任意阶段返回首页，目标和产物可恢复。
- 每个目标独立清洗、独立分析、独立追溯。
